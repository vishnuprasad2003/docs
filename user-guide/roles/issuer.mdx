# Issuer Guide

This comprehensive guide covers all aspects of the Intain Markets platform for borrowers and issuers, from initial setup to ongoing facility management.

## Overview

As an issuer, you have access to:

- **Term Sheet Creation** - Create and manage term sheets
- **Master Commitment Review** - Review auto-generated master commitments
- **Funding Request Management** - Submit and manage funding requests
- **Funding Notice Processing** - Handle funding notices and token management
- **Loan Mapping** - Map loans to credit facilities
- **Document Management** - Upload and manage documents

**Account Setup:**

**Platform Access:**

## Term Sheet Management

### Creating Term Sheets

**Step 1: Access Creation**
1. Navigate to **"Create Term Sheet"**
2. Select **"New Term Sheet"**
3. Choose facility type
4. Begin term sheet creation

**Step 2: Complete Details**
- **Facility Information** - Basic facility details
- **Financial Terms** - Interest rates, amounts, terms
- **Collateral Information** - Collateral details
- **Supporting Documents** - Required documentation

**Step 3: DocuSign Process**
1. **Review Document** - Review term sheet details
2. **Initiate DocuSign** - Start electronic signature process
3. **Complete Signature** - Sign electronically
4. **Submit for Review** - Submit to facility agent

<Image src="/images/termSheet Creation.png" alt="Term Sheet Creation Interface" />

### Term Sheet Status Management

**Status Workflow:**
```
Draft → BorrowerSigned → FAReview → Accepted ✅
```

**Status Descriptions:**
- **Draft** - Initial creation, fully editable
- **BorrowerSigned** - After DocuSign completion
- **FAReview** - Submitted for facility agent review
- **Accepted** - Approved by facility agent

## Master Commitment Process

### Auto-Generation

**Automatic Process:**
- Master commitments are **auto-generated** when term sheets are approved
- No manual creation required
- Pre-populated with term sheet data
- Status starts as **Draft**

**Review Process:**
1. **Notification** - Receive notification of auto-generation
2. **Review Details** - Review master commitment details
3. **Monitor Progress** - Track facility agent progress
4. **Await Activation** - Wait for lender approval

### Master Commitment Status

**Status Progression:**
```
Draft → PendingLenderApproval → ACTIVE
```

**Key Points:**
- **Draft** - Facility agent configuring details
- **PendingLenderApproval** - Waiting for lender approval
- **ACTIVE** - Available for funding requests

## Funding Request Process

### Creating Funding Requests

**Prerequisites:**
- Master commitment status must be **ACTIVE**
- All required documentation ready
- Borrowing capacity available

**Step 1: Access Creation**
1. Navigate to **ACTIVE** master commitment
2. Click **"Create Funding Request"**
3. Select **"New Funding Request"**
4. Begin request creation

**Step 2: Complete Request**
- **Request Amount** - Amount to borrow
- **Purpose** - Detailed purpose description
- **Collateral Description** - Collateral details
- **Supporting Documents** - Required documentation

**Step 3: Submit for Review**
1. **Review Details** - Verify all information
2. **Submit Request** - Submit for facility agent review
3. **Monitor Status** - Track review progress
4. **Respond to Changes** - Address any change requests

<Image src="/images/fundingRequest.png" alt="Funding Request Interface" />

### Funding Request Status

**Status Workflow:**
```
DRAFT → FAReview → APPROVED ✅
```

**Status Descriptions:**
- **DRAFT** - Initial creation, fully editable
- **FAReview** - Submitted for facility agent review
- **APPROVED** - Approved by facility agent

## Funding Notice Management

### Auto-Generation Process

**Automatic Generation:**
- Funding notices are **auto-generated** when funding requests are approved
- No manual creation required
- Pre-populated with funding request data
- Status starts as **PENDING_ESIGN**

**Process Steps:**
1. **Notification** - Receive notification of generation
2. **DocuSign Process** - Complete electronic signature
3. **Token Creation** - FT tokens created automatically
4. **Token Approval** - Approve token transfers

### Funding Notice Status

**Status Progression:**
```
PENDING_ESIGN → ESIGN_COMPLETED → ISSUED
```

**Key Actions:**
- **PENDING_ESIGN** - Complete DocuSign process
- **ESIGN_COMPLETED** - Approve token transfers
- **ISSUED** - Visible to lenders

## Loan Mapping

### Mapping Process

**Step 1: Access Mapping**
1. Navigate to **ACTIVE** master commitment
2. Click **"Map Loans to Facility"**
3. Select loans to map
4. Complete mapping process

**Step 2: Loan Selection**
- **Individual Selection** - Choose specific loans
- **Bulk Selection** - Select multiple loans
- **Filter Options** - Use filters to narrow selection
- **Validation** - Ensure loan eligibility

**Step 3: Complete Mapping**
1. **Review Selection** - Verify selected loans
2. **Submit Mapping** - Complete mapping process
3. **Monitor Status** - Track mapping progress
4. **Verify Results** - Confirm successful mapping
