# Facility Agent Guide

This comprehensive guide covers all aspects of the Intain Markets platform for facility agents and underwriters, from term sheet review to ongoing facility management.

## Overview

As a facility agent, you have access to:

- **Term Sheet Review** - Review and approve term sheets
- **Master Commitment Management** - Configure and manage master commitments
- **Funding Request Review** - Review and approve funding requests
- **Facility Oversight** - Monitor facility performance
- **Lender Management** - Manage lender relationships
- **Document Management** - Access and manage all documents

## Term Sheet Review Process

### Review Workflow

**Review Steps:**
1. **Receive Notification** - Get notified of new term sheet
2. **Review Details** - Examine term sheet information
3. **Make Decision** - Approve, reject, or request changes
4. **Communicate Decision** - Provide feedback to issuer

**Review Criteria:**
- **Financial Terms** - Interest rates, amounts, terms
- **Collateral Quality** - Collateral assessment
- **Borrower Creditworthiness** - Credit analysis
- **Market Conditions** - Current market conditions

### Review Actions

#### Approve Term Sheet

**Approval Process:**
1. **Review All Details** - Comprehensive review
2. **Verify Documentation** - Check all documents
3. **Assess Risk** - Evaluate risk factors
4. **Click Approve** - Approve term sheet

**Result:**
- Status changes to **Accepted**
- Master commitment auto-generated
- Issuer notified of approval

#### Reject Term Sheet

**Rejection Process:**
1. **Identify Issues** - Identify rejection reasons
2. **Provide Feedback** - Detailed rejection feedback
3. **Click Reject** - Reject term sheet
4. **Communicate Decision** - Notify issuer

**Result:**
- Status changes to **Rejected**
- Final state - no further actions
- Issuer can create new term sheet

#### Request Changes

**Change Request Process:**
1. **Identify Changes** - Specify required changes
2. **Provide Details** - Detailed change requirements
3. **Set Deadline** - Set resubmission deadline
4. **Click Request Changes** - Submit change request

**Result:**
- Status changes to **CHANGES_REQUESTED**
- Issuer can edit and resubmit
- Maintains revision history

## Master Commitment Management

### Auto-Generation Process

**Automatic Generation:**
- Master commitments are **auto-generated** when term sheets are approved
- Pre-populated with term sheet data
- Status starts as **Draft**
- Ready for facility agent configuration

**Configuration Steps:**
1. **Review Pre-populated Data** - Verify term sheet data
2. **Configure Rules** - Set collateral eligibility rules
3. **Add Lenders** - Configure lender groups
4. **Finalize Commitment** - Complete configuration

<Image src="/images/facilityAgentCreditFacilityDashboard2.png" alt="Facility Agent Dashboard 2" />

### Master Commitment Configuration

**Step 1: Collateral Rules**
- **Rule Groups** - Create rule groups
- **Eligibility Criteria** - Set eligibility criteria
- **Validation Rules** - Configure validation rules
- **Risk Parameters** - Set risk parameters

**Step 2: Lender Groups**
- **Lender Information** - Add lender details
- **Commitment Amounts** - Set commitment amounts
- **Voting Percentages** - Configure voting percentages
- **Contact Information** - Add contact details

**Step 3: Finalization**
1. **Review Configuration** - Verify all settings
2. **Validate Rules** - Ensure rule validity
3. **Submit for Approval** - Submit to lenders
4. **Monitor Progress** - Track approval progress

### Master Commitment Status

**Status Progression:**
```
Draft → PendingLenderApproval → ACTIVE
```

**Status Management:**
- **Draft** - Configuration in progress
- **PendingLenderApproval** - Waiting for lender approval
- **ACTIVE** - Available for funding requests
