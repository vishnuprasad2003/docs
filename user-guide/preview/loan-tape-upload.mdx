## Loan Tape Upload

Loan tapes contain loan-level data that will be used for verification and potential securitization.

## Preparing Your Loan Tape

### File Requirements

**File Format**: Excel (.xlsx or .xls)

**Required Information**:
- Loan ID (unique identifier)
- Borrower information
- Loan amount
- Interest rate
- Loan term
- Origination date
- Property details (if applicable)

### Template Download

Download the platform loan tape template:
1. Navigate to **Loan Tapes** section
2. Click **Download Template**
3. Use template for data entry

## Uploading Loan Tapes

### Upload Process

1. Navigate to your preview pool
2. Click **Upload Loan Tape**
3. Select your Excel file
4. Review upload summary
5. Confirm upload

### Data Validation

**System Validates**:
- File format (Excel only)
- Required fields present
- Data types correct
- Data completeness

**Validation Errors**:
- System identifies errors
- Provides error details
- Allows corrections
- Resubmit after fixes

## Field Mapping

### Automatic Mapping

The platform automatically maps common fields:
- Loan ID
- Borrower Name
- Loan Amount
- Interest Rate
- Term
- Origination Date

### Custom Mapping

For non-standard fields:
1. Review field mapping
2. Map columns to fields
3. Save mapping
4. Preview mapped data

## Data Standardization

### Standard Field Names

The platform uses standard field names for consistency:
- `LoanID`
- `BorrowerName`
- `LoanAmount`
- `InterestRate`
- `LoanTerm`
- `OriginationDate`

### Mapping to Standards

<Warning>
Ensure all fields are properly mapped to standard field names for accurate processing.
</Warning>

## Reviewing Uploaded Data

### Data Preview

View uploaded loan data:
- Loan count summary
- Data quality metrics
- Field completeness
- Validation status

### Making Corrections

If errors found:
1. Download error report
2. Correct data in Excel
3. Re-upload corrected file
4. Verify corrections

## Best Practices
