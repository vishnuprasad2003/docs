## NFT Minting Overview

Once loans are verified, the platform mints blockchain NFTs (Non-Fungible Tokens) to represent ownership and enable transfer to external platforms like BDB.

## NFT Creation Process

### Automatic Minting

For approved pools:
1. System creates NFT for each loan
2. NFT includes loan metadata
3. Stored on blockchain
4. Ownership recorded

### NFT Metadata

**Included Information**:
- Loan ID
- Loan details
- Borrower information
- Loan terms
- Verification status
- IPFS document links

## NFT Structure

### Blockchain Record

Each NFT contains:
- Unique token ID
- Owner address
- Metadata URI (IPFS)
- Verification proof
- Transaction history

### IPFS Storage

**Stored Documents**:
- Loan application
- Verification certificate
- Supporting documents
- Payment records

## Transfer to BDB

### Transfer Process

1. Generate transfer file
2. Export verified loans
3. Submit to BDB platform
4. Confirm transfer
5. Update NFT ownership

### Transfer File Format

**File Contents**:
- Loan IDs
- Verification status
- Transfer details
- Timestamps
- Signatures

## NFT Management

### Viewing NFTs

**Information Displayed**:
- NFT ID
- Loan details
- Owner information
- Verification status
- Metadata hash

### NFT Ownership

**Ownership Records**:
- Blockchain ownership
- Transfer history
- Current owner
- Verification status
