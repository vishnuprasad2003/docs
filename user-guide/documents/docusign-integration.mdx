# DocuSign Integration

The Intain Markets platform integrates with DocuSign to provide electronic signature capabilities for all credit facility documents. This guide covers the complete DocuSign workflow.

## Overview

DocuSign integration enables:

- **Electronic Signatures** - Sign documents digitally
- **Workflow Automation** - Automated signature workflows
- **Status Tracking** - Real-time signature status updates
- **Document Security** - Secure document handling
- **Audit Trail** - Complete signature history

## DocuSign Workflows

### Term Sheet DocuSign

**Trigger:** When term sheet status is `Draft`

**Process:**
1. Borrower initiates DocuSign process
2. DocuSign envelope created automatically
3. Borrower receives email invitation
4. Borrower completes electronic signature
5. Status changes to `BorrowerSigned`

**Result:** Term sheet status changes to `BorrowerSigned`

### Master Commitment DocuSign

**Trigger:** When master commitment status is `PendingLenderApproval`

**Process:**
1. Any selected lender can initiate DocuSign
2. DocuSign envelope created with lender details
3. Lender receives email invitation
4. Lender completes electronic signature
5. Status changes to `ACTIVE`

**Result:** Master commitment becomes `ACTIVE` and available for funding requests

### Funding Request DocuSign

**Trigger:** When funding request status is `FAReview`

**Process:**
1. Facility agent reviews funding request
2. Facility agent approves via DocuSign
3. DocuSign envelope created automatically
4. Facility agent completes electronic signature
5. Status changes to `APPROVED`

**Result:** Funding request approved and funding notice auto-generated

### Funding Notice DocuSign

**Trigger:** When funding notice is auto-generated with `PENDING_ESIGN` status

**Process:**
1. Borrower receives DocuSign invitation
2. Borrower reviews funding notice details
3. Borrower completes electronic signature
4. Status changes to `ESIGN_COMPLETED`
5. FT tokens created automatically

**Result:** Funding notice signed and FT tokens created

### Step 2: Access Document

- Platform reflects new status
- Next steps initiated

**Cannot Access Document:**
- Check email for DocuSign invitation

- Check spam/junk folders
