## Uploading Documents

Throughout the credit facility workflow, you'll need to upload various documents. This guide explains how to upload files and manage your documents.

---

## Supported File Types

### Document Formats

<CardGroup cols={2}>
  <Card title="PDF Files" icon="file-pdf">
    Preferred format for official documents
  </Card>
  <Card title="Excel Files" icon="file-excel">
    For loan tapes and data files (.xlsx, .xls)
  </Card>
  <Card title="Word Documents" icon="file-word">
    For text documents (.docx, .doc)
  </Card>
  <Card title="Images" icon="image">
    For supporting visuals (.jpg, .png)
  </Card>
</CardGroup>

### File Size Limits

| File Type | Maximum Size |
|-----------|-------------|
| **Documents (PDF, Word)** | 50MB |
| **Data Files (Excel, CSV)** | 10MB |
| **Images** | 10MB |

<Note>
**Best Practice:** Keep files under 10MB when possible for faster uploads and processing.
</Note>

## How to Upload Files

### Basic Upload Process

<Steps>
  <Step title="Navigate to Upload Area">
    Go to the section where you need to upload (Term Sheet, Funding Request, etc.)
  </Step>
  <Step title="Click Upload">
    Find and click the "Upload" or "Add Document" button
  </Step>
  <Step title="Select Your File">
    Choose a file from your computer or drag and drop into the upload area
  </Step>
  <Step title="Wait for Processing">
    The system will validate and process your file
  </Step>
  <Step title="Confirm Upload">
    You'll see a success message when the upload is complete
  </Step>
</Steps>

### Upload Methods

<Tabs>
  <Tab title="Browse Files">
    1. Click "Browse" or "Select File"
    2. Navigate to your file on your computer
    3. Select the file and click "Open"
  </Tab>
  <Tab title="Drag and Drop">
    1. Open the folder containing your file
    2. Drag the file over to the upload area
    3. Drop it when the area highlights
  </Tab>
  <Tab title="Multiple Files">
    1. Select multiple files at once (Ctrl+Click or Cmd+Click)
    2. Drag all files to the upload area
    3. Or use the file browser to select multiple
  </Tab>
</Tabs>

## Document Types in the Workflow

### Term Sheet Documents

When creating a term sheet, you may need to upload:

- **Financial statements**
- **Supporting documentation**
- **KYC documents**

### Funding Request Documents

For funding requests, common uploads include:

- **Collateral addendum**
- **Updated financials**
- **Supporting materials**

### Servicer Documents

Servicers upload:

- **Monthly loan tapes**
- **Performance reports**
- **Servicing documentation**

## After You Upload

### Automatic Validation

The system automatically checks your uploads:

| Check | What It Verifies |
|-------|-----------------|
| **Format** | File is in a supported format |
| **Size** | File is within size limits |
| **Security** | File is scanned for issues |
| **Content** | File is readable and valid |

### Viewing Your Uploads

Once uploaded, you can:

- **View** the document within the platform
- **Download** a copy to your computer
- **See history** of all uploads
- **Track status** of document processing

## Tips for Successful Uploads

### Before Uploading

<AccordionGroup>
  <Accordion title="Prepare Your File" icon="file-check">
    - Ensure the file is complete and correct
    - Use PDF format for official documents
    - Check file size is within limits
    - Use a descriptive filename
  </Accordion>
  
  <Accordion title="Check Your Connection" icon="wifi">
    - Use a stable internet connection
    - Avoid uploading over slow connections
    - Don't interrupt uploads in progress
  </Accordion>
  
  <Accordion title="Organize Your Documents" icon="folder">
    - Keep local backups of uploaded files
    - Use consistent naming conventions
    - Note which documents you've uploaded
  </Accordion>
</AccordionGroup>

## Troubleshooting

<Accordion title="Upload Failed" icon="circle-xmark">
**Common Causes:**
- File too large
- Unsupported format
- Network issue

**Solutions:**
- Check file size limits
- Convert to PDF if needed
- Check your internet connection
- Try uploading again
</Accordion>

<Accordion title="Upload Stuck" icon="spinner">
**Common Causes:**
- Slow internet
- Large file size
- Browser issue

**Solutions:**
- Wait - large files take time
- Check internet connection
- Try a different browser
- Refresh and try again
</Accordion>

<Accordion title="Wrong File Uploaded" icon="triangle-exclamation">
**What to Do:**
- Contact support to assist
- Upload the correct file
- Note which was the error
</Accordion>

<Accordion title="Can't Find My Document" icon="magnifying-glass">
**What to Do:**
- Check the correct section
- Look in document history
- Verify upload completed
- Contact support if needed
</Accordion>

## File Security

Your uploaded files are protected:

- **Encrypted** during transfer and storage
- **Access controlled** based on your role
- **Permanently stored** for audit purposes
- **Version tracked** if you upload updates
